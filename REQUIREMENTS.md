# Требования к проекту Game Organization Bot

## Дата создания: 5 февраля 2026

## Описание
Телеграм бот для организации игр (футбол, волейбол и т.д.)

## Технологический стек

### Backend
- **Runtime**: Node.js
- **Язык**: TypeScript
- **База данных**: PostgreSQL

### Инфраструктура
- **Контейнеризация**: Docker
- **Оркестрация**: Docker Compose
- **Окружения**: 
  - Dev (разработка) - локальная БД
  - Prod (продакшн) - отдельная БД

## Запуск
Локально с компьютера

## Следующие шаги
- Обсуждение и проектирование функционала бота
- Разработка основных команд
- Реализация логики организации игр
